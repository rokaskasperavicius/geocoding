# Geocoding

## Requirements

- Node >= v14.x.x (Recommended: v14.18.0)
- NPM >= v6.x.x (Recommended: v6.14.15)

## Setup

1. `npm install`
3. `npm start`

## APIs
1. Weather - https://openweathermap.org/
2. Location search - https://locationiq.com/
3. Map Tiles - https://locationiq.com/ and https://www.maptiler.com
